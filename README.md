# 掲示板アプリ

## 概要
Spring Bootを使用して作成した掲示板アプリです。  
会員登録、ログイン、投稿作成、投稿一覧表示、投稿編集・削除などの基本機能を実装しました。  
学習用プロジェクトとして、Webアプリ開発の基本的な流れを理解することを目的に作成しました。

## 使用技術
- Java 17
- Spring Boot
- Spring Data JPA
- PostgreSQL
- HTML
- JavaScript
- Lombok
- Maven

## 主な機能
- 会員登録
- ログイン / ログアウト
- 投稿作成
- 投稿一覧表示
- 投稿詳細表示
- 投稿編集 / 削除
- 検索機能（タイトル・作成者）
- ページネーション

## 工夫した点
- Controller、Service、Repository の役割を分けて実装しました
- Entity をそのまま返さず、DTO を使ってデータを受け渡ししました
- パスワードは暗号化して保存するようにしました
- セッションを使ってログイン状態を管理できるようにしました
- 投稿者本人のみ編集・削除できるように制御しました

## 学習したこと
このプロジェクトを通して、Spring Boot における基本的な構成と、CRUD処理の流れを学ぶことができました。  
また、ログイン機能やセッション管理、DB連携を実装しながら、Webアプリケーション開発の基礎を実践的に経験しました。
